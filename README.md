# Tokyo Night Theme for Prism Launcher

- Copy `Tokyo-Night` folder to `C:/Users/<Your-Name>/AppData/Roaming/PrismLauncher/themes` folder.
- Open **Prism Launcher**, go to **Settings** then under **User Interface** tab, set **Widgets** to **Tokyo Night**.
